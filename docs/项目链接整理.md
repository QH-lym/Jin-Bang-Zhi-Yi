## 八、建议统一规则

1. 对外提交的正式 URL 全部使用 `http://jinbangzhiyi.online/` 域名。
2. 安装包 URL 使用英文小写、连字符和版本目录，例如：`/downloads/v1.0.0/jin-bang-zhi-yi-setup-1.0.0.exe`。
3. 页面 URL 保持简洁固定：`privacy.html`、`support.html`、`installer-exit-codes.html`。
4. 本地调试链接统一保留 `http://localhost:3000`。
5. CloudBase 端点不要填到应用商店“软件包 URL”字段；该字段应填写直接下载 `.exe` 或 `.msi` 的正式站点地址。
