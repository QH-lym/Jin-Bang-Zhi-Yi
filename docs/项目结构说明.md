# 晋梆智绎 - 项目结构说明

> 三晋非遗数字化赋能平台 | 面向晋剧与三晋非遗场景的数字化展示与互动平台

## 技术栈概览

| 模块 | 技术 |
|------|------|
| Web 前端 | React 18 + TypeScript + Vite 5 + Tailwind CSS + Framer Motion |
| 3D 渲染 | Three.js |
| 本地数据 | Dexie / IndexedDB |
| 云端能力 | 腾讯云 CloudBase (JS SDK + 云函数) |
| 桌面端 | Electron 42 + electron-builder (NSIS) |
| 移动端 | Capacitor 8 Android |
| 独立后端 | Node.js / Express (阿里云 OSS 存储) |
| 部署 | IIS、CloudBase 静态托管、Docker/Nginx |

## 常用命令

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建 Web 版本
npm run build

# 构建 Electron 桌面端
.\scripts\build-electron.ps1

# 构建 Android 版本
.\scripts\build-android.ps1

# 部署到 IIS
.\scripts\deploy-iis.ps1

# 启动后端服务
.\scripts\start-server.ps1
```
