Installer Exit Codes Documentation | 版本 v1.0.0 | 更新日期 2026-05-11

说明：本应用使用 Electron Builder 生成 Windows x64 NSIS 安装包，不是 MSI 安装包。 因此 Microsoft Partner Center 中应填写本项目定义的 NSIS 返回代码，不使用 MSI 的 1602、1603、1638、3010 等代码。

✅ Microsoft Partner Center 填写值

表单字段 建议填写 说明
用户取消了安装 1 NSIS 默认返回码：用户点击取消或关闭安装器
应用程序已存在 3 检测到相同版本已安装时返回
安装正在进行中 2 Electron Builder 会阻止多个安装器实例同时运行，NSIS 脚本中止返回 2
磁盘空间已满 5 安装前检测到目标磁盘可用空间不足时返回
需要重启 9 检测到 Windows 存在待重启标记时返回
其他安装失败场景文档 URL https://jinbangzhiyi.qd.je/installer-exit-codes.html 本页面作为安装返回代码说明文档

📋 本项目安装包返回代码

返回代码 场景描述 说明
0 ✓ 安装成功 应用程序已成功安装到系统
1 ⚠ 用户取消安装 NSIS 默认返回码：用户在安装过程中点击取消或关闭安装窗口
2 ✗ 安装脚本中止 / 安装正在进行中 NSIS 默认返回码：安装脚本主动中止，例如检测到另一个安装器实例
3 ⚠ 应用程序已存在 自定义 NSIS 返回码：检测到相同版本已经安装
5 ✗ 磁盘空间已满 自定义 NSIS 返回码：目标磁盘空间不足
9 ⚠ 需要重启 自定义 NSIS 返回码：系统存在待重启任务
6710886 ✗ 无法提升权限 Electron Builder NSIS 特定返回码，十六进制为 0x666666

🔧 NSIS 安装器特定代码

返回代码 场景描述 说明
0 ✓ 成功 NSIS 安装成功完成
1 ⚠ 用户取消 NSIS 官方默认返回码：Installation aborted by user
2 ✗ 脚本中止 NSIS 官方默认返回码：Installation aborted by script
0x666666 / 6710886 ✗ 无法提升权限 Electron Builder 模板中的权限提升失败返回码

🚫 不适用于本项目的 MSI 代码

代码 含义 本项目处理方式
1602 MSI 用户取消安装 本项目改用 NSIS 返回码 1
1603 MSI 安装期间致命错误 本项目不是 MSI 安装包；通用脚本中止使用 2
1638 MSI 已安装另一个版本 本项目改用自定义 NSIS 返回码 3
3010 MSI 安装成功但需要重启 本项目改用自定义 NSIS 返回码 9
