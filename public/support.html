关于晋梆智绎

晋梆智绎是一款基于现代Web技术栈开发的晋剧文化数字化展示与互动平台，旨在通过沉浸式数字体验，传承和推广山西晋剧这一国家级非物质文化遗产。

平台融合了3D可视化、实时互动、云端数据同步等前沿技术，为用户提供全方位的晋剧文化探索体验。

主要功能

  • 3D脸谱工坊：交互式3D晋剧脸谱展示，支持旋转、缩放
  • 汉服租赁：在线浏览、预订汉服，支持到店取衣或快递配送
  • 文创商城：脸谱面具、戏曲周边、定制礼品购买
  • 云端同步：本地数据自动备份到云端，多设备无缝切换
  • 数据可视化：销售流向图、运营仪表盘
  • AI对话：智能问答，晋剧知识咨询

技术架构

前端技术

  • React 18 + TypeScript：组件化开发，类型安全
  • Three.js：3D可视化渲染
  • Tailwind CSS：原子化CSS样式
  • Framer Motion：流畅动画效果

后端服务

  • 腾讯云 CloudBase：无服务器架构，云函数 + 云数据库
  • IndexedDB：浏览器本地数据存储
  • HTTPS：全站加密传输

常见问题 (FAQ)

Q1: 数据存储在哪里？

您的数据首先存储在浏览器本地（IndexedDB），确保离线可用。开启同步功能后，数据会加密传输并存储在腾讯云服务器上。

Q2: 如何同步数据到云端？

登录后，系统会自动在后台同步您的数据。您也可以进入「管理后台」→「同步腾讯云」手动触发同步。

Q3: 支持哪些浏览器？

推荐使用 Chrome、Edge、Firefox、Safari 等现代浏览器。IE 浏览器不支持。

Q4: 如何租赁汉服？

进入「汉服租赁」页面 → 选择心仪款式 → 选择租赁日期 → 填写联系信息 → 提交订单 → 等待确认。

Q5: 订单如何支付？

目前支持微信支付、支付宝等多种支付方式。具体支付方式以结算页面显示为准。

Q6: 忘记密码怎么办？

请联系管理员重置密码。目前版本暂不支持自助找回密码功能。

Q7: 如何导出我的数据？

进入「用户设置」→「数据管理」→「导出数据」，可将您的本地数据导出为 JSON 文件。

系统要求

  • 操作系统：Windows 10/11、macOS 10.14+、Linux、iOS、Android
  • 浏览器：Chrome 90+、Edge 90+、Firefox 88+、Safari 14+
  • 网络：建议宽带或4G/5G网络
  • 存储空间：浏览器需至少 50MB 可用空间
